
# RGR ML Dashboard

Streamlit-приложение для инференса моделей машинного обучения и визуализации данных на наборе Breast Cancer Wisconsin (Diagnostic).

## Состав проекта

- `train_models.py` — обучение моделей, сравнение по метрикам и сохранение артефактов.
- `app.py` — Streamlit-дешборд с 4 вкладками:
  - разработчик,
  - информация о датасете,
  - визуализации,
  - предсказание по CSV или вручную.
- `artifacts/` — метрики, лучшие модели, preprocessor.
- `models/` — сериализованные модели.
- `data/breast_cancer_sample.csv` — пример данных для проверки.

## Запуск

```bash
pip install -r requirements.txt
python train_models.py
streamlit run app.py
```

## Примечания

- В отчёте можно заменить плейсхолдеры ФИО, группы и фото на свои данные.
- Для загрузки CSV в приложении используйте те же признаки, что и в обучающем наборе.
- Модели XGBoost и CatBoost сохраняются встроенными методами соответствующих библиотек; sklearn-модели — через pickle.
